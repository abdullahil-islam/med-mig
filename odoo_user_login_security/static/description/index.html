Secure your Odoo from Intruders with User Login Security

Have you ever considered business loss due to data theft?


According to cyber security ventures prediction, cybercrime will cost around 10.5 trillion annually by 2025.


Data theft is primarily connected with the theft of bank details. However, the data theft of a business can lead to several serious problems. It includes the theft of client information, identity theft, intellectual property theft, sabotage, resource theft, and many more.


Safeguard Odoo Against Brute-Force Attack

Authorizing multiple login attempts is one of the biggest threats. It's the call of the time to secure your Oodo account from brute-force attacks.


The User login security module permits you to configure the limit of failed login attempts. And you can also specify the lockout time when a user reaches the maximum limit of failed login attempts.

Minimizes the Chances of Cyber Attacks

Preventing multiple login sessions simultaneously to reduce the risk of account hacking to a great extent. Also, it directly stops attackers from getting into your account if you are already logged in.Accomplishing the same for your Odoo account is feasible with user login security in Odoo.


Do you have a Habit of Leaving the System Logged In?


It's necessary to log out of your account once you leave your system. If not, then anyone else can misuse it. The odoo app facilitates you to set the session expiry time limit. Hence, your Odoo account will be logged out automatically after a preset time of inactivity. You can also set the background actions that you don't want to consider in the activeness of the session.

Session Monitoring Details via User Login Security

The module session details include:


  • Session ID
  • User
  • Login Time
  • Logout Duration
  • Remote IP
  • Operating System
  • Browser
  • Browser Version

Suspicious Activity Alert

Tracking suspicious login activities is crucial. It allows you to keep track of whether your account is secure or not. Additionally, you can end the unrecognized sessions. And receive mail on suspicious logins.


Log out from Suspicious Logins


The Odoo app allows you to change your account password as per choice from the Odoo backend. And you can log out from all other sessions together or simply log out from a particular one by session ID.

Better Visualization from Dashboard with Session Information

Get session details with the help of data cards and pie charts. Scroll left to right and vice versa to track all the sessions in the browsers. Read the data cards to know recently active logins, which include browser, user, IP address, and date.


Pie charts show the session by status, which shows filters like logged in, logged out, terminated, time out, and login failed.

(New) Improved Security with Two-Factor Authentication

The module now has 2fa for fortified protection against unauthorized login. You must provide a 6-digit login OTP every time you log in to your account. Also, you can implement session management for two-factor authentication.

Prerequisite for Odoo User Login Security

For using two-factor authentication, you need to install the Google Authenticator app on your mobile device to use the authentication service.
